### プロダクト名

Footics

### ターゲットユーザー

- サッカー戦術に興味のあるファン
- サッカーを深く観戦したい人
- 試合中のメモや分析を残したい人

### 解決する課題

- 試合を見ながら戦術的なポイントを記録したいが、適切なツールがない
- 試合後に「あの場面どうだったか」を振り返りたいが記憶が曖昧
- 自分の分析を体系的に蓄積できない

### Phase 1 のゴール

リアルタイムで試合を見ながら、選手ごとのアクションを素早く IndexDB に記録できる MVP を完成させる

### やりたいこと

## テンプレート管理画面

チームのテンプレート情報を登録できる

## 試合画面

試合(Match) / 設定(Setup) / 履歴(History)を切り替えられる

# Match

アクションを記録できる

# Setup

試合情報 選手情報を設定できる

# History

記録したアクションをタイムラインとして確認できる

## 機能

# Match

Timer 機能(TimerWidget): アクションの時間を記録するためのタイマー
フォーメーション変更機能: ピッチに表示されるチームのフォーメーションを変更できる
アクション選択機能: ピッチにフォーメーション通りに表示された選手を選択してボトムシートからアクションを選択できる
直近アクション編集機能: 直近に記録したアクションを編集できる

# Setup

試合情報登録機能: チーム 会場 日時を登録できる
選手情報登録機能: 選手を登録できる
